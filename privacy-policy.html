개인정보 처리방침

마지막 업데이트: 2024년 7월 29일

업무용 To-Do List 확장 프로그램 개인정보 처리방침

1. 개인정보 수집 및 이용

본 확장 프로그램은 사용자의 개인정보를 수집하지 않습니다. 모든 데이터는 사용자의 로컬 브라우저에만 저장되며, 외부 서버로 전송되지 않습니다.

2. 수집하는 데이터

  • 사용자가 직접 입력한 할 일 목록
  • 할 일의 중요도 및 시급도 설정
  • 완료된 작업 기록
  • 사용자 설정 (테마, 정렬 방식 등)

3. 데이터 저장 위치

모든 데이터는 사용자의 로컬 브라우저 스토리지에만 저장됩니다. 외부 서버나 클라우드 서비스에는 저장되지 않습니다.

4. 데이터 처리 방식

  • 로컬 스토리지 사용으로 완전한 개인정보 보호
  • 오프라인에서도 완벽 작동
  • 사용자가 언제든지 데이터 삭제 가능
  • 자동 데이터 백업 기능 없음

5. 제3자 데이터 공유

본 확장 프로그램은 사용자 데이터를 제3자와 공유하지 않습니다. 모든 데이터는 사용자의 로컬 환경에서만 처리됩니다.

6. 데이터 보안

  • 브라우저의 기본 보안 기능 활용
  • 외부 네트워크 통신 없음
  • 사용자 데이터 암호화 없음 (로컬 저장으로 불필요)

7. 사용자 권리

  • 언제든지 확장 프로그램을 제거하여 모든 데이터 삭제 가능
  • 브라우저 설정에서 로컬 스토리지 데이터 직접 삭제 가능
  • 데이터 내보내기 기능 없음

8. 쿠키 및 추적 기술

본 확장 프로그램은 쿠키나 추적 기술을 사용하지 않습니다.

9. 연령 제한

본 확장 프로그램은 모든 연령대의 사용자가 이용할 수 있습니다.

10. 개인정보 처리방침 변경

본 방침은 필요시 변경될 수 있으며, 변경사항은 확장 프로그램 업데이트를 통해 공지됩니다.

11. 문의처

개인정보 처리에 관한 문의사항이 있으시면 개발자에게 직접 연락해 주시기 바랍니다.

본 확장 프로그램은 사용자의 개인정보를 최대한 보호하기 위해 최소한의 데이터만 로컬에 저장하며, 외부 서버와의 통신을 하지 않습니다.
